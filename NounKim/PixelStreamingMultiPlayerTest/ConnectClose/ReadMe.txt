픽셀 스트리밍 테스트 파일입니다.

1. WindowsNoEditor 폴더에 들어가서 config.json - 설정 파일 들어가기(메모장으로 열면 됨)
2. ChangeCollab 주소 가져오기
3. config.json의 urPath에 주소 붙여넣기, 단 폴더를 구분하는 단락에 / 가 아니라 // 를 하도록 한다.
4. 웹페이지 주소창에 로컬 IP 주소 쳐보기

주의사항:

시판되는 그래픽카드는 동시 인코딩이 최대 3개까지로 제한이 걸려있음. 따라서 그래픽카드 제한을 따로 해제해주지 않는 이상
4번 항목을 4번 이상 실행할 수 없다.

수정되지 않은 오류:

엔진에서 Null 인코딩 생성 제한만 주석으로 해제해놓았음. 따라서 인코딩 수가 늘어남에 따라 치직거리고 동기화가 안되는 문제가 발생함.
수령님 축지법 쓰신다!
