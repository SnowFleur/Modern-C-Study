nDisplay의 구동을 위한 준비물

UE4 4.27 버전이 필요하다.



구동 방법
1. 4.27 버전에서 플러그인 셋팅 
2. nDisplay 플러그인과 SwitchBoard 플러그인을 Enabled로 바꾼다.
3. 재시작한다.
4. 재시작하면 스위치보드 아이콘이 상단 작업표시줄에 뜬다.
5. 스위치 보드 아이콘 옆에 있는 삼각형을 눌러 리스너를 켠다.
6. 스위치보드 아이콘을 누른다.
7. 스위치보드가 켜지면 Configs -> new config -> 첨부한 config.png 파일 참고
8. sequence 아래쪽에 있는 Add Device -> nDisplay 
9. config2.png 참고하여 NDT.uasset을 Browse 한다.

이후는 config3를 참고하라.
